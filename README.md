# LuX-AI-Season-2

Lux AI Season 2 - NeurIPS Stage | Team Martian

## How to run the code

1. Install Python 3.10
1. Install pdm
1. Run the following commands

    ```bash
    pdm venv create
    eval $(pdm venv activate in-project)
    pdm sync
    ```

1. Open [`tutorial.ipynb`](tutorial.ipynb) and start coding!
